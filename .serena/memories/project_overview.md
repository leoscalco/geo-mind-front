GeoMind Front is a Next.js 16.2.1 + React 19 + TypeScript 5 frontend for a geopolitical intelligence platform. Main UI entry is app/page.tsx, which preloads dashboard and financial snapshot data and renders the client component NexusDashboard. Relevant folders: app/ (App Router entrypoints), components/dashboard/ (main product UI), components/ui/ (small UI primitives), lib/ (api client, utils, demo fixtures), types/ (DTO mirrors from backend).
